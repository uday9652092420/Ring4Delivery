🚀 Getting Started

Step 1: Open the Application

• Double-click index.html to open directly in your browser

• OR use a local server (recommended):

python -m http.server 8000
# Then visit http://localhost:8000

Step 2: Explore the Home Page

• View the hero section with welcome message

• Click "Start Order" button to proceed

• Read about why customers choose FoodHub

Step 3: Browse Restaurants (Page 2)

• Search restaurants by name

• Filter by cuisine type (Italian, Chinese, Indian, American, Mexican)

• Click "View Menu" on any restaurant

Italian
Chinese
Indian

Step 4: Select Items from Menu (Page 3)

• View categorized menu items (Appetizers, Main Course, Sides, Desserts)

• Click "Add" on any item

• Set quantity and add special instructions if needed

• Click "Add to Cart"

Step 5: Review Your Cart (Page 4)

• View all items with images and pricing

• Adjust quantities using +/- buttons

• Remove items if needed

• See order summary with:

  • Subtotal
  • Delivery Fee (₹40)
  • Tax (5%)
  • Total Amount

Step 6: Complete Checkout

• Click "Proceed to Checkout"

• Enter delivery details:

  • Full Name
  • Email Address
  • Mobile Number
  • Delivery Address
  • Special Notes (optional)

Step 7: Send Order via WhatsApp

• Click "Place Order via WhatsApp"

• Your order details will be sent to:

+62 81407746

• You'll receive order confirmation on WhatsApp

• Order includes: Order ID, Customer Info, Items, Pricing

⚙️ Customization Guide

Change WhatsApp Number:

Edit cart.html, find line with const recipientPhone = '6281407746'

Add More Restaurants:

Edit js/data.js, add objects to restaurants array

Add Menu Items:

Edit js/data.js, add objects to menus array

Change Colors:

Edit css/style.css, modify CSS variables in :root
